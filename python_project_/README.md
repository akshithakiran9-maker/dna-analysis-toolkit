# DNA Analysis Toolkit 

A simple Python-based bioinformatics script for analyzing DNA sequences.

##  Features
- DNA sequence validation
- Nucleotide counting (A, T, G, C)
- GC content calculation
- Reverse complement generation
- DNA to protein translation
- Motif (pattern) detection
- Mutation identification between sequences

##  Technologies Used
- Python (basic scripting)
## ▶️ How to Run

```bash
python dna_toolkit.py

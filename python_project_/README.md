 BIOINFORMATICS TOOLKIT USING PYTHON(v1.0)
---
A bioinformatics toolkit built in Python to perform fundamental DNA sequence analysis, inspired by real-world genomic data processing workflows.

---
 What This Project Does

This tool allows users to input DNA sequences and automatically performs multiple analyses commonly used in bioinformatics:

- Validates DNA sequences (A, T, G, C)
- Calculates nucleotide composition and GC content
- Generates reverse complement sequences
- Translates DNA into protein sequences
- Identifies repeating motifs (k-mers)
- Detects mutations between two sequences

---
 How It Works

The script processes raw DNA input step-by-step:

1. Cleans and validates the sequence  
2. Computes statistical properties (length, base counts, GC%)  
3. Applies biological transformations (complement, translation)  
4. Scans for repeating patterns  
5. Compares sequences for mutation detection
